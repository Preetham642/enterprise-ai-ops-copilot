# Enterprise AI Operations Copilot

A production-style AI workflow application built using FastAPI, OpenAI, LangGraph, ChromaDB, and Retrieval-Augmented Generation (RAG).

## Features

- FastAPI REST APIs
- OpenAI Integration
- ChromaDB Vector Search
- OpenAI Embeddings
- Retrieval-Augmented Generation (RAG)
- LangGraph Workflow Orchestration
- Service Status Tool
- Incident Ticket Creation Tool
- Semantic Search
- Enterprise Knowledge Base

## Workflow

User Query
↓
Document Retrieval
↓
Service Health Check
↓
Incident Ticket Creation
↓
AI Response Generation

- Python
- FastAPI
- OpenAI
